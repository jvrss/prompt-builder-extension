# prompt-builder-extension

Build prompts in parts, concatenate, copy, and inject into ChatGPT

![image link](https://raw.githubusercontent.com/jvrss/prompt-builder-extension/refs/heads/main/docs/ui.png)

The send to chatgpt button does not work yet, but you can copy the prompt and paste it manually.

The plugin has memory to store the prompts created, even after closing the browser.

## Installation

1. Download the repository on your computer.
2. Open Google Chrome and go to `chrome://extensions/`.
3. Enable "Developer mode" by toggling the switch in the top right corner.
4. Click on "Load unpacked" and select the folder of repository.
5. The extension should now be installed and visible in your browser toolbar.
